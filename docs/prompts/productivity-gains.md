# Generate a Productivity Gains Stair Step Diagram

Use the /microsim-generator skill and the p5.js guide to create a new microsim in @docs/sims/productivity-jumps It is a stair step interactive infographic with the the label productivity on the left vertical axis and several step that the user clicks on.  The baseline (manual) is the lowest level which is 3,000 person hours to create an intelligent textbook.  The next step up is using ChatGPT to generate individual pages and elements.  That step cuts the hours to 1,000 hours.  The second step is Claude Code, that cuts the time down to 100 hours. The third step is Claude Code Skills.  That cuts the total time down to 10 hours to generate an entire textbook.  When the user clicks on a step, an infobox that describes the skill in detail appears below the step diagram.
