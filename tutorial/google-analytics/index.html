    Configuring Google Analytics

    In the mkdocs.yml template file you will find the following lines:

    1
2
3
4
    #extra:
#  analytics:
#    provider: google
#    property: G-XXXX

    You will need to first generate a Google Analytics property ID and then remove the comment lines "#".

    1
2
3
4
    extra:
  analytics:
    provider: google
    property: G-XXXX
